# Contributing

Ce dépôt est personnel mais suit une discipline simple.

## Convention de commits

- `init:` création de la structure
- `study:` progression sur un thème
- `notes:` ajout ou amélioration de synthèses
- `quiz:` ajout de questions / réponses
- `review:` révision d’un sujet
- `fix:` correction d’erreurs ou ambiguïtés
- `exam:` simulation, résultats, analyse d’écarts

## Convention de branches

- `feat/togaf-p1-weekX`
- `feat/togaf-p2-scenarios`
- `feat/archimate-l1-core`
- `feat/archimate-l2-practice`
- `chore/restructure`
- `review/monthly-retrospective`

## Définition de terminé

Une session est considérée utile si elle produit au moins un des éléments suivants :
- une synthèse personnelle
- une fiche erreur
- une carte mentale / tableau comparatif
- un mini-cas modélisé
- une mise à jour du tracker
