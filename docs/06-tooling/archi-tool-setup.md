# Outillage de modélisation

## Outil recommandé pour pratiquer
Archi (outil de modélisation ArchiMate)

## Usage conseillé
- créer un modèle par grand cas d’étude
- créer des dossiers par couches :
  - Business
  - Application
  - Technology
  - Motivation
  - Implementation & Migration
- ne pas commencer par des vues trop grandes
- privilégier des vues ciblées par stakeholder

## Routine
1. modéliser un premier brouillon
2. vérifier la sémantique
3. simplifier
4. seulement ensuite améliorer la lisibilité

## Export
Conserver :
- source du modèle
- image exportée
- note expliquant l’objectif de la vue
