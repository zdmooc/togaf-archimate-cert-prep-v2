# Weekly reviews

Créer un dossier par semaine :

- week-01
- week-02
- week-03
- ...

Chaque semaine doit contenir :
- `summary.md`
- `mistakes.md`
- `actions.md`

## Contenu minimal

### summary.md
- ce que j’ai compris
- ce qui reste fragile
- ce que je peux déjà expliquer à voix haute

### mistakes.md
- erreurs de concepts
- erreurs de lecture des questions
- erreurs de modélisation

### actions.md
- ce que je reprends lundi
- ce que je diffère
- quelle certification est la plus mûre
