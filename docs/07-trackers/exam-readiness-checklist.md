# Exam readiness checklist

## TOGAF Part 1
- [ ] je connais les 6 blocs du syllabus
- [ ] je peux expliquer l’ADM simplement
- [ ] je distingue deliverable / artefact / building block
- [ ] je connais le rôle de la gouvernance
- [ ] j’ai fait au moins 2 simulations

## TOGAF Part 2
- [ ] je sais lire un scénario lentement
- [ ] je sais éliminer les réponses plausibles mais moins pertinentes
- [ ] je sais quand utiliser l’open book
- [ ] j’ai fait au moins 3 simulations scenario-based

## ArchiMate niveau 1
- [ ] je connais les couches principales
- [ ] je reconnais les éléments majeurs
- [ ] je connais les relations fondamentales
- [ ] j’ai modélisé au moins 3 mini-cas

## ArchiMate niveau 2
- [ ] je sais choisir une vue adaptée à un stakeholder
- [ ] je sais justifier un choix de modélisation
- [ ] je maîtrise mieux les scénarios et la pertinence des vues
- [ ] j’ai fait au moins 3 exercices comparatifs

## Logistique examen
- [ ] compte exam prêt
- [ ] pièce d’identité vérifiée
- [ ] environnement de passage prêt
- [ ] date réservée
