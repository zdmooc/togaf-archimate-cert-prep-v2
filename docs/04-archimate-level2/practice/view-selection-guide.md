# ArchiMate niveau 2 — guide de sélection de vue

## Si l’audience est sponsor métier
Préférer :
- vue simple
- peu d’éléments
- accent sur valeur, services, transformation

## Si l’audience est architecture applicative
Préférer :
- vue des composants, services, dépendances
- liens explicites avec métier si nécessaire

## Si l’audience est plateforme / ops
Préférer :
- vue technology
- dépendances d’hébergement
- flux ou relations de support pertinentes

## Si l’audience est gouvernance de transformation
Préférer :
- implementation and migration
- work packages
- liens avec cible et transitions

## Règle
Ne pas choisir la vue la plus “riche”.
Choisir la vue la plus utile.
