# ArchiMate niveau 2 — Banque de scénarios

## Scénario 1 — Sponsor métier
On vous demande une vue simple expliquant comment une transformation améliore l’expérience client.

## Scénario 2 — Equipe applicative
On vous demande de clarifier quelles applications supportent quels services métier.

## Scénario 3 — Equipe plateforme
On vous demande une vue montrant la dépendance entre composants applicatifs et socle d’hébergement.

## Scénario 4 — Comité d’architecture
On vous demande une vue de migration par lots.

## Scénario 5 — Sécurité / IAM
On vous demande une vue de dépendances autour d’un service d’authentification central.

## Scénario 6 — Rationalisation
On vous demande une vue permettant d’identifier redondances et consolidation possibles.

Pour chacun :
- audience
- message
- viewpoint plausible
- couches dominantes
- éléments minimaux
- erreurs à éviter
