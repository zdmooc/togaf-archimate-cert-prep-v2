# ArchiMate niveau 2 — Syllabus map

## Ce que le niveau 2 ajoute
- application pratique du langage
- scénarios plus riches
- choix du meilleur modèle
- usage plus fin des relations
- mécanismes avancés
- customization mechanisms
- cohérence globale du modèle

## Ce qu’il faut entraîner
- lire le besoin métier implicite
- sélectionner les bons éléments
- éliminer les relations incorrectes
- justifier pourquoi une modélisation est meilleure qu’une autre
- équilibrer précision et lisibilité

## Révision active
Pour chaque cas :
1. qui lit la vue ?
2. quel est l’objectif ?
3. quel niveau de détail ?
4. quelle couche / cross-layer ?
5. quels éléments minimaux ?
6. quelles relations strictement nécessaires ?
