# ArchiMate niveau 2 — checklist de revue de modèle

## Pertinence
- [ ] la vue répond à une vraie question
- [ ] le stakeholder cible est identifiable
- [ ] le niveau de détail est cohérent

## Sémantique
- [ ] les éléments choisis sont corrects
- [ ] les relations sont valides
- [ ] aucune relation vague n’est utilisée par défaut

## Lisibilité
- [ ] la vue n’est pas surchargée
- [ ] les regroupements sont visibles
- [ ] le sens global est compréhensible en moins de 30 secondes

## Utilité
- [ ] la vue aide à décider, comprendre ou communiquer
- [ ] elle pourrait être montrée en comité ou en atelier
- [ ] elle peut être reliée à une action réelle
