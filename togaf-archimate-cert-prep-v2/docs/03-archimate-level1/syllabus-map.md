# ArchiMate niveau 1 — Syllabus map

## Objectif
Maîtriser les bases de la notation, de la structure et des relations de la langue de modélisation ArchiMate.

## Sujets à couvrir
- concepts de base et terminologie
- structure du langage
- generic metamodel
- relations de base
- Motivation
- Strategy
- Business Layer
- Application Layer
- Technology Layer
- Physical elements
- cross-layer modeling
- Implementation and Migration
- viewpoint mechanism

## Approche recommandée
1. apprendre les familles d’éléments
2. apprendre les relations fondamentales
3. modéliser des cas simples
4. vérifier la cohérence sémantique avant la beauté du schéma

## Cas pratiques suggérés
- vue métier d’un traitement de sinistre
- vue applicative d’une chaîne décisionnelle IBM ODM
- vue technologique d’une plateforme OpenShift
- vue implémentation d’une migration VM -> conteneurs
