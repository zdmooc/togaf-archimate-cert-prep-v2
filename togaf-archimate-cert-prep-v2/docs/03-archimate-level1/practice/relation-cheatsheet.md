# ArchiMate niveau 1 — pense-bête relations

## À savoir reconnaître rapidement
- composition
- aggregation
- assignment
- realization
- serving
- access
- triggering
- flow
- influence
- association

## Exercice type
Pour chaque relation :
1. expliquer en mots simples
2. donner un exemple métier
3. donner un exemple applicatif
4. donner un contre-exemple

## Erreur typique
Utiliser association quand une relation plus précise existe.
